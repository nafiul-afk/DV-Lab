# Data Visualization Lab

A personal repository for lab codes and practice problems from my **Data Visualization** course. Each class folder contains hands-on implementations, exercises, and experiments based on the course syllabus.

---

##  Tech Stack

- **Python** (primary language)
- **Google Colab** / Jupyter Notebooks
- **NumPy**, **Pandas** — data manipulation
- **Matplotlib**, **Seaborn** — static visualizations
- **Plotly** — interactive visualizations
- **Scikit-learn** — clustering (k-means, k-medoids), PCA, t-SNE, UMAP
- **Tableau** — business intelligence & dashboards

##  Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/nafiul-afk/DV-Lab.git
   cd DV-Lab
   ```

2. Install dependencies:
   ```bash
   pip install numpy pandas matplotlib seaborn plotly scikit-learn wordcloud umap-learn
   ```

3. Open any notebook in **Google Colab** or **Jupyter**:
   ```bash
   jupyter notebook
   ```
